## 安装 go-sql-driver/mysql package

go 有一个包 `database/sql`, 这个包封装了所有的数据库，但是没有包含
数据库驱动.

这个地方就需要下载一个 数据库驱动.
```shell
go get -u github.com/go-sql-driver/mysql
```
